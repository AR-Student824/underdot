blobby underdot.js

a lightweight javascript library that adds to existing functions to make developing easier. not as stupid and heavy as jquery, don't worry.

[]

Things to keep in mind

don't combine with similar libraries

other libraries may interfere with underdot's functionality, which can result in broken code.

features from future js versions

features from future js versions that replace underdot's functionality shall be removed or renamed asap. you will be warned if this ever happens.
